[i]Были добавлены следующие горячие клавиши:[/i]

[b][color="#F77F00"]F[/color][/b] — перевести фокус на поле ввода сообщения, чтобы можно было сразу начать печатать без клика мышью.
[b][color="#F77F00"]Esc[/color][/b] — снять фокус с поля ввода.

[b]Все последующие клавиши работают только когда фокус на поле ввода отсутствует:[/b]

[b][color="#F77F00"]U[/color][/b] — показать или скрыть список пользователей. Работает как для чата, так и для чатлога.
[b][color="#F77F00"]B[/color][/b] — открыть список банов. Если он уже открыт — вернуться обратно в чат.
[b][color="#F77F00"]P[/color][/b] — открыть управление произношением никнеймов. Если уже открыто — вернуться в чат.
[b][color="#F77F00"]M[/color][/b] — включить или выключить звуковые эффекты.
[b][color="#F77F00"]V[/color][/b] — включить или выключить голосовое озвучивание сообщений.
[b][color="#F77F00"]N[/color][/b] — переключить режим уведомлений по циклу: Stack → Replace → Muted.
[b][color="#F77F00"]T[/color][/b] — включить или выключить режим "поверх всех окон".
[b][color="#F77F00"]R[/color][/b] — сбросить размер и положение окна до значений по умолчанию.
[b][color="#F77F00"]C[/color][/b] — изменить цвет своего никнейма. [b][color="#F77F00"]Ctrl+C[/color][/b] — сбросить цвет. [b][color="#F77F00"]Shift+C[/color][/b] — обновить цвет с сервера.
[b][color="#F77F00"]X[/color][/b] — выйти из приватного режима.
[b][color="#F77F00"]Space[/color][/b] — прокрутить вниз на одну страницу.
[b][color="#F77F00"]Shift + Space[/color][/b] — прокрутить вверх на одну страницу.

[b]Следующие клавиши работают всегда, независимо от фокуса:[/b]

[b][color="#F77F00"]Ctrl + ;[/color][/b] — открыть / закрыть панель смайликов.
[b][color="#F77F00"]Ctrl + U[/color][/b] — сменить аккаунт.
[b][color="#F77F00"]Ctrl + T[/color][/b] — переключить тему оформления (светлая / тёмная).

[b]Vim-style прокрутка — работает в чате и чатлогах:[/b]

[b][color="#F77F00"]J[/color][/b] — прокрутить вниз.
[b][color="#F77F00"]K[/color][/b] — прокрутить вверх.
[b][color="#F77F00"]gg[/color][/b] — перейти в начало (двойное нажатие G с интервалом до 300 мс).
[b][color="#F77F00"]Shift + G[/color][/b] — перейти в конец.

[b]Дополнительные клавиши в режиме чатлогов:[/b]

[b][color="#F77F00"]M[/color][/b] — включить / выключить фильтр упоминаний (вместо звуковых эффектов).
[b][color="#F77F00"]P[/color][/b] — открыть / закрыть парсер чатлогов.
[b][color="#F77F00"]D[/color][/b] — открыть выбор даты.
[b][color="#F77F00"]S[/color][/b] — открыть / закрыть поиск. Когда парсер открыт — запускает парсинг.
[b][color="#F77F00"]Esc[/color][/b] — закрыть поиск если открыт.
[b][color="#F77F00"]H[/color][/b] — предыдущий день (поддерживает удержание для быстрой перемотки).
[b][color="#F77F00"]L[/color][/b] — следующий день (поддерживает удержание для быстрой перемотки).
[b][color="#F77F00"]C[/color][/b] — отменить парсинг (когда идёт парсинг).
[b][color="#F77F00"]Ctrl + F[/color][/b] — открыть / закрыть поиск (работает независимо от фокуса).
[b][color="#F77F00"]Ctrl + P[/color][/b] — открыть чатлог и парсер из любого места.
[b][color="#F77F00"]Ctrl + C[/color][/b] — скопировать результаты парсинга (когда парсер открыт).
[b][color="#F77F00"]Ctrl + S[/color][/b] — сохранить результаты парсинга в файл (когда парсер открыт).

[b]Горячие клавиши окна управления аккаунтами:[/b]

[i]Раздел Connect:[/i]

[b][color="#F77F00"]Tab[/color][/b] — переключить аккаунт в выпадающем списке.
[b][color="#F77F00"]Enter / E[/color][/b] — подключиться к чату.
[b][color="#F77F00"]A[/color][/b] — перейти к созданию аккаунта.
[b][color="#F77F00"]D[/color][/b] — удалить выбранный аккаунт.
[b][color="#F77F00"]C[/color][/b] — изменить цвет никнейма. [b][color="#F77F00"]Ctrl+C[/color][/b] — сбросить. [b][color="#F77F00"]Shift+C[/color][/b] — обновить с сервера.
[b][color="#F77F00"]1[/color][/b] — переключить Auto-login. [b][color="#F77F00"]2[/color][/b] — Start minimized. [b][color="#F77F00"]3[/color][/b] — Start with system.

[i]Раздел Create:[/i]

[b][color="#F77F00"]Tab[/color][/b] — переключить фокус между полями Username и Password.
[b][color="#F77F00"]Enter[/color][/b] — сохранить аккаунт (когда поле ввода в фокусе).
[b][color="#F77F00"]Ctrl + S[/color][/b] — сохранить аккаунт.
[b][color="#F77F00"]Esc[/color][/b] — вернуться в раздел Connect.

[b]Горячие клавиши не зависят от раскладки клавиатуры.[/b]
